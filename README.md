# Personal Finance Tracker

A simple command-line tool to help you track your personal income and expenses.

## Features
- Add new income or expense transactions.
- View your complete transaction history.
- Check your current account balance.
- All data is saved automatically to a local file (`transactions.txt`).

## How to Run
1. Make sure you have Python 3 installed on your computer.
2. Clone or download this repository.
3. Open your terminal/command prompt and navigate to the project folder.
4. Run the program with the following command:
   ```bash
   python finance_tracker.py
